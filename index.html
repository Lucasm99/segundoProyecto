HTML 5 Y CSS 3

HTML 5

El 28 de Octubre de 2014 se aprobo la Recomendacion HTML 5, culminando asi diez año de trabajo. En 2004 se formo el WHATWG, un grupo formal de individuos y empresa decididos a impulsar el desarrollo del HTML, que se encontraba bloqueado en el W3C desde el año 2000. En 2007 el W3C retomo el trabajo sobre el HTML y siete año despues se alcanzo el objetivo.

Pero el HTML no era mas que el primer paso. En noviembre de 2016 se aprobo la recomendacion HTML 5.1, ampliando y clasificando el HTML 5, en diciembre de 2017 se aprobo la recomendacion HTML 5.2 y el W3C prepara el HTML 5.3 para finales del 2018.

CSS 3

Desde la aprobacion de CSS 2 en 1998, el W3C ha estado trbajando en la siguente generacion de las hojas de estilo, las CSS 3. Las recomendaciones CSS 3 se planificaron como un conjunto de recomendaciones en vez de como un documento unico. El proceso resulto mucho mas largo delo esperado: las primeras recomendaciones CSS 3 no se aprobaron hasta el 2011.

Aun que los navegadores admiten muchas caracteristicas de los borradores de CSS 3, la publicacion de HTML 5 no ha acelerado el desarrollo de CSS3.
